## City Top 5 Restaurants

A single-page site with a search input. Type a city, get back the top 5 restaurants with name, cuisine, short description, and price range, generated by Lovable AI (Gemini) with structured JSON output.

### Pages
- `/` (`src/routes/index.tsx`) — hero with title, subtitle, search input, results list, loading and error states.

### Backend
- `src/routes/api/top-restaurants.ts` — TanStack server route. POST `{ city }`. Validates with zod (1–100 chars). Calls Lovable AI Gateway via `@ai-sdk/openai-compatible` + `ai` (`generateText` with `Output.object`) using `google/gemini-3-flash-preview`. Returns `{ city, restaurants: [{ name, cuisine, priceRange, description }] }`. Handles 429/402 with friendly errors.
- `src/lib/ai-gateway.ts` — shared provider helper (Lovable-API-Key + X-Lovable-AIG-SDK headers).

### Frontend behavior
- React Query mutation calls the API on submit.
- Renders 5 restaurant cards (shadcn Card + Badge for cuisine/price).
- Skeletons while loading, toast on error.
- Empty state before first search.

### Design
- Clean modern look using existing tokens. Warm neutral background, large display heading, single accent. Inter-style system font is fine; no extra deps.
- SEO: route `head()` with title "Top 5 Restaurants in Any City" and description.

### Dependencies
- Add: `ai`, `@ai-sdk/openai-compatible`, `zod` (if not present).

### Out of scope
- No auth, no database, no history, no maps. (Easy to add later.)
